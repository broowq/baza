# БАЗА → Stitch — пакет промптов для генерации экранов

**Проект Stitch:** «БАЗА — Cinematic Glass» — `projects/3683633692604861832`
**Дизайн-система (применяется по умолчанию):** «Cinematic Glass» — `assets/624868cef721481ca9942ff3d6f3661f`

Дизайн-система уже залита и выбрана в проекте, поэтому каждый промпт ниже описывает
**только контент и структуру экрана** — стиль (тёмное стекло, мятный акцент, белые
pill-кнопки, ультралайт-Geist, glass-карточки) Stitch берёт из дизайн-системы.

> Как пользоваться: открой проект в Stitch (web/app) → «New screen» / «Generate» →
> вставь промпт → выбери Desktop (или Mobile где указано) + дизайн-систему
> «Cinematic Glass» + модель Gemini 3.1 Pro. Генерация идёт на стороне Stitch
> (а не через здешний MCP, где рвётся соединение).

## 1. Лендинг (Desktop)
Маркетинговый лендинг «БАЗА» — B2B-лидогенерация. Весь UI на русском. Секции сверху вниз:
sticky blurred top-nav («база» + мятный аватар-тайл слева; pill-ссылки «Продукт», «Тарифы», «Кейсы» по центру; ghost «Войти» + белая pill «Начать бесплатно» справа); герой по центру на near-black поле с мятно-небесными радиальными свечениями и маскированной hairline-сеткой (mono-eyebrow «B2B ЛИДОГЕНЕРАЦИЯ»; огромный ультралайт-заголовок Geist-200 «Лиды, которые созревают сами» со словом «созревают» в serif-italic; muted-подзаголовок; белая pill «Начать бесплатно» с мятным свечением + glass-pill «Смотреть демо»); плавающий glass-фрейм с мокапом дашборда (sidebar + таблица лидов со score-барами и пастельными бейджами); ряд из 3 glass-карточек «Находим», «Обогащаем», «Оцениваем»; KPI-полоса из трёх огромных ультралайт-чисел с mono-подписями; прайс-тизер (три карты Starter/Pro/Business, у «Pro» мятное свечение и pill «ПОПУЛЯРНЫЙ»); финальный CTA-блок + минимальный футер.

## 2. Вход (Mobile)
Экран входа «БАЗА»: центрированная тёмная glass-карточка на near-black поле, вверху «база» + мятный аватар-тайл, заголовок «С возвращением», поля «Email» и «Пароль» (glass-инпуты), белая pill-кнопка «Войти» с мятным свечением, ссылка «Забыли пароль?» (muted), внизу «Нет аккаунта? Зарегистрироваться». Русский UI.

## 3. Регистрация (Mobile)
Экран регистрации «БАЗА»: центрированная glass-карточка, заголовок «Создать аккаунт», поля «Имя», «Email», «Пароль», «Название организации», белая pill «Зарегистрироваться», mono-подпись о согласии с условиями, внизу «Уже есть аккаунт? Войти». Русский UI.

## 4. Дашборд — Проекты (Desktop)
Главный экран продукта «БАЗА». Слева фиксированный 240px glass-сайдбар: «база» + аватар, чип тарифа «Pro», поле поиска + колокол уведомлений, pill-навигация «Проекты»(active, мятная точка), «Лиды», «Аналитика», «Задачи», «Рассылки», «Тарифы», «Настройки»; внизу аватар пользователя + email (mono) + «Выйти →» + переключатель темы. Основная область: заголовок «Проекты» + белая pill «Новый проект»; сетка glass-карточек проектов, в каждой — название, mono-метаданные (создан, лидов), тонкий score-бар, пастельные бейджи статусов. Русский UI.

## 5. Проект — лиды (Desktop)
Детальный экран проекта в «БАЗА». 240px glass-сайдбар (как выше). Контент: хлебные крошки + название проекта; ряд pill-табов «Таблица», «Воронка», «Источники»; тулбар с поиском, фильтрами-pill и белой pill «Собрать лиды»; большая таблица лидов (mono uppercase tracked-caps заголовки на surface-1, hairline-строки, зебра): колонки Компания, Контакт, Телефон, Score (тонкий мятно-янтарный бар + число), Статус (пастельный бейдж с точкой), Источник. Сверху строка из 3-4 stat-tile (всего лидов, с телефоном %, средний score) — ультралайт-числа с мятным радиальным свечением в углу. Русский UI.

## 6. Лиды — общий хаб (Desktop)
Сквозной экран всех лидов организации «БАЗА». Сайдбар (пункт «Лиды» active). Контент: заголовок «Все лиды» + глобальный поиск; ряд фильтров-pill (статус, проект, источник, есть телефон/почта); сетка lead-карточек (glass, равной высоты): название компании (medium white), muted-подзаголовок, пастельный бейдж статуса + точка, тонкий score-бар, внизу прижатые контакты (email + телефон, валидные — мятным); hover приподнимает карту с мятной рамкой. Русский UI.

## 7. Карточка лида — drawer (Desktop)
Десктоп-экран «БАЗА» с открытым правым slide-over drawer (420px, glass, elev-3) поверх затемнённой таблицы лидов. В drawer: шапка с названием компании + score-ring (мятная дуга) + кнопка закрытия; вкладки «Обзор», «Переписка», «Заметки»; в «Переписка» — единый таймлайн коммуникаций (исходящие email справа, входящие ответы слева, отметки каналов и mono-таймстемпы); внизу композер сообщения с белой pill «Отправить». Русский UI.

## 8. Аналитика (Desktop)
Аналитический дашборд «БАЗА». Сайдбар («Аналитика» active). Контент: заголовок «Аналитика» + сегментный переключатель периода (pill-группа); ряд из 4 stat-tile (ультралайт-числа, mono-подписи, дельты мятным/розовым); воронка конверсии (горизонтальные бары, выделенный шаг светится зелёным); тепловая карта активности (мятная сетка); бабл-чарт источников; таблица топ-источников со спарклайнами. Тёмное стекло, единственный мятный акцент. Русский UI.

## 9. Задачи (Desktop)
Экран задач «БАЗА». Сайдбар («Задачи» active, розовый счётчик просроченных). Контент: заголовок «Задачи» + pill-фильтры «Все», «Сегодня», «Просрочено», «Выполнено»; список задач (glass-строки): чекбокс (cbox), название, привязка к лиду/компании, срок (mono; просроченные — розовым), пастельный бейдж приоритета; справа белая pill «Новая задача». Русский UI.

## 10. Рассылки (Desktop)
Экран outreach-кампаний «БАЗА». Сайдбар («Рассылки» active). Контент: заголовок «Рассылки» + белая pill «Новая кампания»; сетка/таблица кампаний с метриками (отправлено, открыто %, ответили %) — mono-числа, тонкие прогресс-бары; справа панель композера последовательности писем (шаги цепочки, glass-карточки, поля темы/тела). Русский UI.

## 11. Настройки (Desktop)
Экран настроек «БАЗА». Сайдбар. Контент с вложенным 220px tab-rail слева (пункты «Профиль», «Организация», «Участники», «Тариф и оплата», «API»); справа форма выбранной секции (glass-инпуты h-11 rounded-2xl, лейблы, белая pill «Сохранить»). Показать секцию «Организация»: название, домен, часовой пояс. Русский UI.

## 12. Тарифы (Desktop)
Экран тарифов «БАЗА». Контент по центру: mono-eyebrow «ТАРИФЫ», ультралайт-заголовок «Простые тарифы», сегментный переключатель «Месяц / Год»; три прайс-карты — «Starter» (3 900 ₽), «Pro» (16 900 ₽, мятное свечение + pill «ПОПУЛЯРНЫЙ», приподнята), «Business» (44 900 ₽); в каждой — цена крупным ультралайтом, список фич с мятными буллет-точками, белая pill «Выбрать». Русский UI.

## 13. Админ (Desktop) — опционально
Админ-панель «БАЗА». Сайдбар (пункт «Админ» виден). Контент: заголовок «Администрирование»; ряд stat-tile (организаций, активных подписок, лидов собрано); таблица организаций (mono-заголовки): Название, Владелец, Email, Тариф (бейдж), Создана, действия (ghost-кнопки управления доступом). Русский UI.
