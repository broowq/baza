Вопросы и ответы

Как происходит сбор лидов?

БАЗА использует SearXNG (self-hosted) для поиска по нише и географии, затем фильтрует агрегаторы и нормализует домены. Для каждого домена извлекаются контактные страницы.

Нужен ли API-ключ Bing?

Нет. Bing API — опциональный источник. По умолчанию работает только SearXNG. Bing подключается через .env.

Где хранятся данные?

Всё хранится в вашей PostgreSQL. Мы не передаём данные третьим лицам.

Можно ли деплоить на свой сервер?

Да. Проект полностью dockerized — docker compose up и всё работает.

Готовы заполнить воронку?

Регистрация занимает 30 секунд. Нет кредитной карты.
